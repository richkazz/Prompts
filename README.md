# AI Prompts Collection

A curated collection of high-quality AI prompts for software engineering, code review, documentation, and more.

## Overview

This repository provides a standardized set of prompts designed for modern AI-assisted development workflows. Whether you're auditing a codebase for security, performing a UI/UX review, or documenting complex logic, these prompts help you get consistent and professional results from LLMs like Claude 3.5 Sonnet and GPT-4o.

## How to Use These Prompts

1. Browse the categories below to find a prompt that fits your task.
2. Open the prompt file and copy the content inside the **Prompt** code block.
3. Paste it into your AI-assisted coding tool (Claude Code, Cursor, etc.).
